# Back-End

## Requirements

- Node.js envoirment
- Node modules
    - Socket.io
    - python-shell
- Python 3.10.0
- Python modules
    - transformer
    - tensorflow
    - pandas
    - selenium

### Initialization steps

- npm install on the "Back-end" directory
- install python 3.10.0
- install required modules
- run ./components/ai/setup.py to initialize required files.
- npm run
